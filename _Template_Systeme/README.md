# TEMPLATE SYSTÈME ÉDITORIAL

## Description

Ce template est une **structure vierge, duplicable et scalable** pour créer un système éditorial complet pour n'importe quelle marque, entreprise ou projet.

## Comment utiliser

```bash
# Dupliquer le template
cp -r _Template_Systeme/ Mon_Nouveau_Projet/

# Personnaliser
cd Mon_Nouveau_Projet/
# Modifier les fichiers selon votre projet
```

## Structure incluse

```
├── Input/                    → Sources et ressources
│   ├── 01_Strategie/        → Business, algorithmes, objectifs
│   ├── 02_Identite/         → Équipe, marque, personnalité
│   ├── 03_Cible/            → Avatar, douleurs, parcours
│   ├── 04_Skill/            → Compétences IA
│   ├── 05_Methodes/         → Frameworks (Hormozi, etc.)
│   ├── 06_Temoignages/      → Témoignages clients
│   ├── 07_Actualites/       → News, événements
│   ├── 08_Demonstrations/   → Transcriptions
│   ├── 09_Feedback/         → Retours
│   ├── 10_Veille/           → Concurrence, juridique
│   ├── 11_Ecosysteme/       → Partenaires, communauté
│   └── 12_Produits_Services/→ Offres
├── Output/                   → Livrables
│   └── Reseaux_Sociaux/     → Par plateforme
├── Banque_Contenus/          → Médias
└── .claude/commands/         → Claude Code
```

## Personnalisation requise

1. **Input/02_Identite/** : Ajouter votre équipe et marque
2. **Input/03_Cible/** : Définir votre avatar client
3. **Input/04_Skill/** : Créer votre SKILL personnalisé
4. **Input/01_Strategie/Objectifs/** : Définir vos objectifs

## Compatible avec

- Claude Code (slash commands)
- Tout outil de gestion de contenu
- Équipes distribuées
