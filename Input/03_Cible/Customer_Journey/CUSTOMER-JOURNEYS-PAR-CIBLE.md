## Principe Fondamental

Chaque cible a son propre Customer Journey. La ligne éditoriale doit :
1. Toucher chaque cible **au bon moment**
2. Avec la **bonne information**
3. Sur le **bon canal**
4. Au **bon jour/heure**

---

## CIBLE 1 : Formateurs OF (Organismes de Formation)

### Phase 1 : AWARENESS (Mois 1-2)

**État client** : Ne connaît pas QALIA, ressent la douleur Qualiopi

| Élément | Détail |
|---------|--------|
| **Émotion dominante** | Frustration, fatigue |
| **Questions** | "Est-ce normal de passer autant de temps sur Qualiopi ?" |
| **Contenu prioritaire** | Éducation métier, validation de la douleur |
| **Format optimal** | Posts texte, carousels éducatifs |
| **Timing** | Mardi/Jeudi 8h30 (avant formations) |

**Messages clés** :
- "40h/mois de paperasse Qualiopi. Ça vous parle ?"
- "Les codes RNCP changent sans préavis"
- "Personne pour valider à 23h"

**Objectif** : Faire reconnaître la douleur, créer l'identification

---

### Phase 2 : CONSIDERATION (Mois 2-3)

**État client** : Compare les solutions, évalue les options

| Élément | Détail |
|---------|--------|
| **Émotion dominante** | Curiosité prudente, scepticisme |
| **Questions** | "C'est quoi QALIA ? C'est fiable ? Ça remplace mon expertise ?" |
| **Contenu prioritaire** | Philosophie copilote, différenciation vs ChatGPT |
| **Format optimal** | Posts témoignages anonymisés, vidéos courtes |
| **Timing** | Mardi/Jeudi/Samedi |

**Messages clés** :
- "L'IA comme copilote, pas pilote automatique"
- "Partagé. Pas remplacé."
- "10 jours → 1-2 heures (témoignage anonymisé)"

**Objectif** : Différencier QALIA, rassurer sur la philosophie

---

### Phase 3 : DECISION (Mois 3-4)

**État client** : Prêt à agir, besoin de preuves finales

| Élément | Détail |
|---------|--------|
| **Émotion dominante** | Urgence, besoin de confiance |
| **Questions** | "Ça vaut 297€/mois ? ROI réel ?" |
| **Contenu prioritaire** | Témoignages avec chiffres, ROI, garanties |
| **Format optimal** | Études de cas anonymisées, comparatifs |
| **Timing** | Tout jour, réactivité importante |

**Messages clés** :
- "Pour moins de 300€, une ingénierie complète"
- "Une ingénierie en mangeant un carry"
- "Trois weekends en famille récupérés"

**Objectif** : Convertir, lever les dernières objections

---

### Phase 4 : RETENTION (Mois 4-6)

**État client** : Client actif, utilise QALIA

| Élément | Détail |
|---------|--------|
| **Émotion dominante** | Satisfaction, découverte continue |
| **Questions** | "Comment optimiser mon usage ? Quoi de neuf ?" |
| **Contenu prioritaire** | Tips avancés, mises à jour, communauté |
| **Format optimal** | Posts behind the scenes, nouveautés |
| **Timing** | Moins fréquent, qualité > quantité |

**Messages clés** :
- "Nouvelle fonctionnalité disponible"
- "Ce que font les meilleurs utilisateurs"
- "Le Cercle Pionnier (offert après acquisition)"

**Objectif** : Fidéliser, approfondir l'usage

---

### Phase 5 : ADVOCACY (Mois 6+)

**État client** : Fan, prêt à recommander

| Élément | Détail |
|---------|--------|
| **Émotion dominante** | Fierté, enthousiasme |
| **Questions** | "Comment partager mon expérience ?" |
| **Contenu prioritaire** | Success stories, invitations à témoigner |
| **Format optimal** | Co-création, interviews |
| **Timing** | Événements, moments clés |

**Messages clés** :
- "Merci aux fondateurs"
- "Vos retours construisent QALIA"
- Programme de parrainage (à définir)

**Objectif** : Activer le bouche-à-oreille

---

## CIBLE 2 : Coachs Individuels

### Phase 1 : AWARENESS

**Différences vs OF** :
- Moins de contraintes Qualiopi (souvent non certifiés)
- Besoin plus axé sur structuration et crédibilité
- Plus ouverts à l'innovation

| Élément | Détail |
|---------|--------|
| **Émotion dominante** | Besoin de légitimité |
| **Questions** | "Comment structurer mon offre ?" |
| **Contenu prioritaire** | Structuration, professionnalisation |
| **Timing** | Weekend (plus disponibles) |

**Messages clés** :
- "Structurer une offre de coaching professionnelle"
- "De l'intuition à la méthode"
- "Crédibilité sans certification"

---

### Phase 2-5 : CONSIDERATION → ADVOCACY

Suivent le même schéma que les OF, avec adaptation du message :
- Moins focus Qualiopi, plus focus structuration
- Moins focus audit, plus focus clients
- Même philosophie copilote

## CIBLE 3 : Auditeurs (Future — Mois 3+)

**FAIRE** : Montrer la valeur système, l'intérêt mutuel

| Phase | Contenu | Ton |
|-------|---------|-----|
| **Awareness** | "Et si QALIA était utilisé par les OF ET les auditeurs ?" | Réflexion |
| **Consideration** | "Moins de friction = plus de qualité" | Collaboratif |
| **Decision** | Partenariat, pas vente | Institutionnel |

**Timing d'introduction** : Mois 3 (teaser discret)
