# INPUT — Sources et Ressources

## Structure

```
Input/
├── 01_Strategie/        → Stratégie business, algorithmes, objectifs
├── 02_Identite/         → Équipe, marque, personnalité
├── 03_Cible/            → Avatar, douleurs, parcours client
├── 04_Skill/            → Compétences IA pour génération contenu
├── 05_Methodes/         → Hormozi, Customer Journey, Growth
├── 06_Temoignages/      → Témoignages clients (à anonymiser)
├── 07_Actualites/       → News secteur, événements, tendances
├── 08_Demonstrations/   → Transcriptions et extraits des démos
├── 09_Feedback/         → Retours internes, clients, communauté
├── 10_Veille/           → Concurrence, juridique, inspirations
├── 11_Ecosysteme/       → Partenaires, ambassadeurs, communauté
└── 12_Produits_Services/→ QALIA, formations, assets
```

## Règles d'utilisation

### Ajout de contenu
1. Placer chaque fichier dans le dossier approprié
2. Nommer avec format : `YYYY-MM-DD_Description.md`
3. Taguer le statut : `[BRUT]`, `[TRAITE]`, `[UTILISE]`

### Témoignages
- **TOUJOURS** anonymiser avant utilisation
- Conserver : secteur, durée, problème, résultat
- Supprimer : prénom réel, nom organisme, lieu précis

### Actualités
- `Passees/` : événements terminés, résultats annoncés
- `Presentes/` : en cours, pertinent maintenant
- `Futures/` : annonces à venir, dates clés
- `Secteur_Formation/` : news Qualiopi, RNCP, réglementation

## Pour Claude Code
Ce dossier est la source de vérité pour générer du contenu.
Toujours vérifier les actualités et témoignages récents avant de créer un post.
