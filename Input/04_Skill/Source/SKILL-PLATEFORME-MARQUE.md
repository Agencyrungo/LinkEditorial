## Définition

Une **plateforme de marque** est le socle stratégique qui définit l'identité profonde d'une marque. Elle sert de référence pour toutes les décisions de communication, design et positionnement.

---

## Structure d'une Plateforme de Marque Complète

### 1. VISION

**Question clé** : Quel monde idéal la marque cherche-t-elle à créer ?

| Élément | Description |
|---------|-------------|
| **Énoncé** | Une phrase qui décrit l'état futur souhaité |
| **Horizon** | Long terme (5-10 ans) |
| **Ambition** | Ce qui dépasse l'entreprise elle-même |

**Exemple format** :
> "Un monde où [bénéficiaire] peut [action transformative] sans [obstacle actuel]."

---

### 2. MISSION

**Question clé** : Que fait concrètement la marque pour atteindre cette vision ?

| Élément | Description |
|---------|-------------|
| **Énoncé** | Action quotidienne de l'entreprise |
| **Pour qui** | Cible principale |
| **Comment** | Moyen différenciant |

**Exemple format** :
> "Nous [action] pour [cible] grâce à [différenciation]."

---

### 3. VALEURS

**Question clé** : Quels principes guident les décisions et comportements ?

**Règles** :
- 3-5 valeurs maximum (au-delà = dilution)
- Chaque valeur doit avoir une définition opérationnelle
- Éviter les valeurs génériques (excellence, qualité, innovation)
- Préférer des valeurs distinctives et vérifiables

| Valeur | Définition opérationnelle | Preuve concrète |
|--------|---------------------------|-----------------|
| [Valeur 1] | Ce que ça signifie en action | Comment on le vérifie |
| [Valeur 2] | ... | ... |
| [Valeur 3] | ... | ... |

---

### 4. PROMESSE DE MARQUE

**Question clé** : Quel engagement la marque prend envers ses clients ?

| Élément | Description |
|---------|-------------|
| **Promesse fonctionnelle** | Bénéfice tangible, mesurable |
| **Promesse émotionnelle** | Sentiment, état procuré |
| **Reason to believe** | Pourquoi la marque peut tenir cette promesse |

**Format** :
> "Nous promettons de [bénéfice fonctionnel] pour que vous ressentiez [bénéfice émotionnel], parce que [raison de croire]."

---

### 5. POSITIONNEMENT

**Question clé** : Quelle place unique la marque occupe-t-elle dans l'esprit des clients ?

**Formule de positionnement** :
> "Pour [cible], [marque] est le/la [catégorie] qui [différenciation unique], parce que [preuves]."

| Élément | Contenu |
|---------|---------|
| **Cible** | Qui précisément |
| **Catégorie** | Dans quel univers |
| **Différenciation** | Ce qui rend unique |
| **Preuves** | Ce qui rend crédible |

---

### 9. TAGLINE / SIGNATURE

**Question clé** : Une phrase courte et mémorable qui résume l'essence de la marque.

**Critères** :
- Courte (3-6 mots idéalement)
- Mémorable
- Différenciante
- Intemporelle (pas liée à une campagne)

**Format propositions** :
1. [Proposition 1]
2. [Proposition 2]
3. [Proposition 3]

---

### 10. MANIFESTE (Optionnel)

Un texte court (100-200 mots) qui capture l'essence de la marque de façon émotionnelle.

**Structure** :
- Ouverture provocatrice ou inspirante
- Problème/tension
- Vision/solution
- Invitation/appel

## Processus de Génération

### Étape 1 : Collecte d'informations

Questions à poser :
1. Quelle est l'activité principale ?
2. Qui sont les clients cibles ?
3. Quels problèmes résolvez-vous ?
4. Qu'est-ce qui vous différencie des concurrents ?
5. Quelles sont les valeurs personnelles du fondateur ?
6. Quel est l'historique/l'origine de l'entreprise ?
7. Quelle ambition à 5-10 ans ?
8. Quels mots les clients utilisent pour vous décrire ?

### Étape 2 : Analyse

- Identifier les patterns récurrents
- Détecter les différenciateurs authentiques
- Trouver la tension centrale (problème ↔ solution)
- Identifier la personnalité naturelle

### Étape 3 : Formulation

- Rédiger chaque élément de la plateforme
- Vérifier la cohérence interne
- Tester la distinctivité vs concurrents

### Étape 4 : Validation

- Lire à voix haute : ça sonne vrai ?
- Montrer à quelqu'un qui connaît la marque : ils se reconnaissent ?
- Appliquer à un cas concret : le message reste cohérent ?

## Points de Contrôle Qualité

### Cohérence interne
- [ ] La vision et la mission sont alignées
- [ ] Les valeurs sont démontrables (pas juste proclamées)
- [ ] La promesse est crédible (reason to believe solide)
- [ ] Le positionnement est différenciant ET vrai
- [ ] La personnalité est cohérente avec le ton de voix

### Authenticité
- [ ] Reflète vraiment l'entreprise (pas un idéal déconnecté)
- [ ] Le fondateur se reconnaît
- [ ] Les clients se reconnaîtraient

### Opérationnalité
- [ ] Chaque élément peut guider une décision concrète
- [ ] Applicable à tous les points de contact
- [ ] Transmissible à une équipe
