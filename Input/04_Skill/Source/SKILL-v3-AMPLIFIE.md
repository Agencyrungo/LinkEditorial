## 9. Formules de Hooks

### Hook Factuel (Chiffre + Problème + Solution)

```
[Chiffre] [Unité] de [problème]. Voici comment [solution].
```

**Exemples validés :**
- "40h/mois de paperasse Qualiopi. Voici comment j'en ai récupéré 35."
- "3 heures. C'est le temps qu'il a fallu pour structurer une formation de 20 jours."
- "1 code RNCP périmé = 1 audit raté. L'IA l'aurait détecté."

### Hook Question Provocante

```
[Question qui casse une croyance] ?
```

**Exemples validés :**
- "Et si l'IA ne servait pas à aller plus vite, mais à aller plus loin ?"
- "Vous formez des gens. Mais qui vous forme à documenter ?"
- "Qualiopi : conformité ou performance ? Et si les deux étaient liés ?"

### Hook Personnel / Ancré

```
[Lieu/Moment]. [Observation]. [Lien avec le message].
```

**Exemples validés :**
- "L'Axe Majeur de Cergy. 3 km de perspective. C'est cette vision qui m'a forgé."
- "La Réunion, 10 000 km de Paris. C'est de là que je forme l'océan Indien à l'IA."

### Hook Contre-intuitif

```
[Ce que tout le monde dit]. [Ce que la réalité montre].
```

**Exemples validés :**
- "L'IA va tous nous remplacer, qu'ils disaient. 3 ans plus tard, je forme des humains à être meilleurs avec elle."
- "Plus de paperasse = plus de conformité ? Non. Plus de paperasse = moins de temps pour former."

### Hook Douleur Formateur

```
[Situation reconnaissable]. [Conséquence]. [Ouverture solution].
```

**Exemples :**
- "23h. Tu relis ton déroulé pour la 4ème fois. Personne pour valider. Ça te parle ?"
- "Le mail de l'auditeur est tombé. Dans 3 semaines, c'est le jour J. Tu es prêt ?"
