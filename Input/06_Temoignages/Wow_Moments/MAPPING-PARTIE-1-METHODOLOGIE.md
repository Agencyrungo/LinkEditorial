# Mapping Exhaustif Démos ↔ Douleurs — PARTIE 1 : Méthodologie

## Objectif du Mapping

Connecter CHAQUE extrait émotionnel des démonstrations QALIA à :
1. Le niveau de douleur spécifique (1-4)
2. La douleur exacte de la cartographie
3. L'indicateur Qualiopi concerné
4. L'exploitation éditoriale recommandée

## Grille d'Analyse par Extrait

Pour chaque phrase émotionnelle :

```yaml

source: Demo-YYYY-MM-DD (ligne XXX)
locuteur: Anonymisé / Romu
contexte_demo: Ce qui se passait à ce moment
emotion_primaire: Choc / Soulagement / Joie / Incrédulité / etc.
emotion_secondaire: Si applicable

mapping_douleur:
  niveau: 1-4
  douleur_exacte: "Citation de la cartographie"
  indicateurs_qualiopi: [liste]

exploitation_editoriale:
  type_post: [Identification / Solution / Preuve / Autorité]
  force: ⭐ à ⭐⭐⭐⭐⭐
  cta_adapte: [Engagement / Curiosité / Démo / Cercle]

```
