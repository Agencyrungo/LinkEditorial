## Notes d'Utilisation

### Pour les posts LinkedIn :
- **Utiliser verbatim** les phrases entre guillemets (force authentique)
- **Anonymiser systématiquement** : "Un formateur m'a dit...", "Lors d'une démo..."
- **Contextualiser** : donner juste assez de contexte pour que ça résonne
- **Ne pas surexpliquer** : laisser l'émotion parler

### Pour les carousels :
- Une phrase par slide
- Image sobre qui renforce l'émotion
- Pas de logo/branding excessif

### Pour les vidéos :
- Possibilité de recréer ces moments (avec accord)
- Utiliser les verbatims comme sous-titres
- Ton authentique, pas de mise en scène excessive
