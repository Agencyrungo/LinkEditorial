## Règles de nommage

### Posts
`POST-[NUM]-[DATE]-[TYPE]-[FORMAT].md`
Exemple : `POST-07-2026-01-28-Engagement-Carousel.md`

### Lignes éditoriales
`LIGNE-EDITORIALE-[RESEAU]-[PERIODE].md`
Exemple : `LIGNE-EDITORIALE-LinkedIn-S1-2026.md`

### KPI
`KPI-[RESEAU]-[MOIS]-[ANNEE].md`
Exemple : `KPI-LinkedIn-Janvier-2026.md`

## Workflow de publication

1. **Génération** → Claude Code génère depuis Input
2. **Validation** → Romu valide le contenu
3. **Préparation** → Visuels, carousels créés
4. **Publication** → Post LinkedIn aux horaires optimaux
5. **Suivi** → KPI enregistrés, ajustements

## Checklist pré-publication

- [ ] Hook < 150 caractères
- [ ] ≤3 hashtags niche
- [ ] Témoignages anonymisés
- [ ] Format différent du post précédent
- [ ] Actu/témoignage récent vérifié
- [ ] Ça sonne comme Romu
