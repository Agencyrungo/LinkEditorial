## 4. SPÉCIFICATIONS PAR FORMAT

### Carrousels (26 posts)
- **Slides** : 5-10 maximum
- **Usage** : Éducatif, tutoriels, comparatifs, listes
- **Timing** : Mardi/Jeudi 8h30
- **CTA** : Slide finale avec action claire

### Sondages (12 posts)
- **Options** : 2-4 choix
- **Durée** : 1 semaine
- **Usage** : Engagement communauté, validation hypothèses
- **Timing** : Tous jours (fort engagement)

### Vidéos (6 posts — 1/mois)
- **Durée** : 60-90 secondes
- **Format** : Vertical (9:16)
- **Sous-titres** : Obligatoires
- **Contenu** : Face caméra, démo QALIA, témoignages
- **Timing** : Mercredi ou Vendredi (différenciation)

### Texte + Image (16 posts)
- **Image** : Visuel unique, pas stock
- **Texte** : 800-1500 caractères
- **Usage** : Témoignages, citations, insights
- **Timing** : Flexible

### Texte seul (12 posts)
- **Longueur** : 1000-2000 caractères
- **Usage** : Storytelling, réflexions, expériences personnelles
- **Timing** : Mardi/Jeudi/Samedi matin

## 6. RÈGLES DE NON-RÉPÉTITION V3

1. **Jamais 2 formats identiques consécutifs**
2. **Carrousels espacés d'au moins 1 post**
3. **Sondages espacés d'au moins 4 posts**
4. **Vidéo = événement du mois (positionnement stratégique)**
5. **Alternance thématique : Douleur → Solution → Preuve → Vision**
