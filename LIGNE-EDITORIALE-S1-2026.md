### Cadence de Publication (2026)

| Paramètre | Valeur | Justification |
|-----------|--------|---------------|
| **Fréquence** | 3 posts/semaine | Optimal selon Algorithm Insights 2025 |
| **Jours** | Mardi, Jeudi, Samedi | Peak B2B + créneau week-end moins concurrentiel |
| **Heure** | 10h-12h | Formateurs disponibles, golden hour 2026 |
| **Espacement** | ≥18h entre posts | Évite pénalité spam |
| **Format rotation** | Texte → Carousel → Texte/Vidéo | Évite -35% reach répétition |

## Règles Transversales

### Checklist universelle (chaque post)

- [ ] Un seul message
- [ ] Pas de paradoxe interne
- [ ] Hook < 150 caractères
- [ ] Vocabulaire métier (indicateur, NC, déroulé, RNCP)
- [ ] Formulation positive
- [ ] QALIA naturel (pas pitch)
- [ ] Punch line mémorable
- [ ] ≤3 hashtags niche
- [ ] Pas de demande de likes
- [ ] Répondre aux commentaires dans l'heure

### Interdits absolus

- Présentation produit directe
- Liens dans le corps du post
- "Likez si vous êtes d'accord"
- Plus de 3 emojis
- Jargon startup
- Promesses exagérées
- Formulations négatives

### Méthode storytelling (5 lignes)

Chaque post narratif suit :
1. **Situation** — contexte initial
2. **Désir** — ce que le personnage veut
3. **Conflit** — l'obstacle
4. **Changement** — la solution/action
5. **Résultat** — la transformation
