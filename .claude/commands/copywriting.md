## Les 7 Structures Disponibles

### 1. DOULEUR → ESPOIR (max 25%)
```
[Hook douleur] → [Problème] → [Et si...] → [Ouverture]
```

### 2. VALEUR PURE (25%)
```
[Hook éducatif] → [Contenu actionnable] → [Liste/étapes] → [Takeaway]
```

### 3. TÉMOIGNAGE (20%)
```
[Contexte anonymisé] → [Citation verbatim] → [Résultat] → [Pont audience]
```

### 4. CONTRE-INTUITIF (15%)
```
[Affirmation surprenante] → [Développement] → [Preuves] → [Renforcement]
```

### 5. BEHIND-THE-SCENES (10%)
```
[Accroche personnelle] → [Coulisses] → [Apprentissage] → [Connexion]
```

### 6. QUESTION / ENGAGEMENT (5%)
```
[Question ouverte] → [Contexte minimal] → [Invitation]
```

### 7. STORYTELLING PERSONNEL (10%)
```
[Moment précis] → [Tension] → [Résolution] → [Morale]
```

## Règles

1. **Jamais** deux posts consécutifs avec la même structure
2. Hook < 150 caractères
3. Post 1300-2000 caractères
4. ≤3 hashtags niche
5. Pas de lien externe
